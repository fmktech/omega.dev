# Module: context-bootstrap

## Owns

`src/context/context-service.ts` and `context-service.test.ts`.

## Implements

Implement `ContextService`. Discover regular `AGENTS.md` files without following symlinks or traversing ignored VCS, dependency, generated, and build trees. Bound individual and aggregate instruction bytes, retain scope and SHA, and order instructions from root toward nested scopes.

Return compact project-knowledge and installed-skill catalogs during the pre-model runner handshake. Read a full skill only when its component belongs to the requested project harness and has kind `skill`. Skill content remains immutable in the object store; the catalog is metadata, not an alternate authority.
